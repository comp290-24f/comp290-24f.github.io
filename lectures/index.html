Synopsis

In this lecture, students are introduced to the course, their peers, and course expectations. We introduce students to the shell and the command-line interface. Students learn about running shell programs, environment variables, and filepaths.

Week 2: Files and Shell Scripts

Synopsis

In this lecture, students are introduced to file permissions and hidden files. Students also learn how to use package managers like Homebrew and winget to install external shell programs. As an example, students use the fzf program and explore other scripts uploaded to GitHub.
